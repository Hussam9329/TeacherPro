# إصلاح ظهور ملف الطالب داخل إدارة الفرص

## المشكلة
عند فتح **ملف الطالب** من تبويبة **إدارة الفرص** كان يظهر بحجم غير كامل أو نصف شاشة في بعض الحالات بسبب رسم مكوّن ملف الطالب داخل شجرة الصفحة نفسها، مما يجعله يتأثر بحاويات الأب أو خصائص العرض/التمرير.

## الإصلاح
- جعل `StudentProfileDialog` يُعرض عبر `createPortal` مباشرة داخل `document.body`.
- رفع طبقة العرض إلى `z-[999]` حتى لا يدخل تحت أي نافذة أو تبويب.
- إجبار ملف الطالب على عرض كامل الشاشة باستخدام `h-[100dvh]` و `w-screen`.
- إضافة `role="dialog"` و `aria-modal="true"` لتحسين الدلالة والوصول.
- إضافة زرين واضحين أعلى ملف الطالب:
  - **رجوع**
  - **إغلاق**
- كلا الزرين يغلقان ملف الطالب ويرجعان المستخدم إلى إدارة الفرص بدون إعادة تحميل الصفحة.

## الملفات المعدلة
- `src/components/teacher-pro/student-profile-dialog.tsx`

## ملاحظة
لم يتم تغيير منطق بيانات الطالب أو الفرص أو الدرجات. التعديل فقط لطريقة عرض ملف الطالب والتحكم بإغلاقه.
